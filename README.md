# dgty-cli

## google

```bash
google <search_query>
```

`search_query` type `string`

## twitch

```bash
twitch <username>
```

`username` or empty string for twitch homepage

## youtube

```bash
youtube <search_query>
```

`search_query` or empty string for youtube homepage

## dict

```bash
dict <words>
```

`words` type `string`
